# Resume Project Description

## Project Title
**EV Adoption Analytics & Forecasting Dashboard — India** *(Personal/Portfolio Project)*

## Resume Bullets (choose 2-4 depending on space)

- Built an end-to-end analytics pipeline processing 7,000+ EV registration records across 24 Indian states, implementing a data-cleaning workflow (Python/Pandas + SQL) that resolved duplicate records, standardized inconsistent categorical fields, and treated statistical outliers using the IQR method, improving dataset reliability for downstream analysis.

- Wrote 10+ advanced SQL queries using CTEs, window functions (RANK, LAG, NTILE), and conditional aggregation to deliver state-wise rankings, year-over-year growth analysis, and market-share breakdowns across vehicle categories and manufacturers.

- Designed and validated an ensemble forecasting model (scikit-learn log-linear regression + custom NumPy-implemented damped Holt's trend method) projecting a 5-year growth trajectory with 95% confidence intervals, achieving R² of 0.93 on historical fit.

- Designed a 6-page Power BI dashboard (Executive Overview, State Analysis, Category Breakdown, Infrastructure Gap Analysis, Time Series, and Forecast) with a custom dark theme, 15+ DAX measures, drill-through pages, bookmarks, and synchronized cross-filtering slicers — architected on a star schema for performance.

- Translated analytical findings into 6 stakeholder-ready business recommendations (e.g., identifying the 5 states with the most severe EV-to-charging-station infrastructure gap), demonstrating the ability to convert raw data into actionable strategy.

## One-Line Summary (for a projects section header)
> Designed and built a full-stack analytics solution (SQL + Python + Power BI) analyzing India's EV adoption trends and forecasting 5-year growth, translating data into actionable infrastructure and policy recommendations.
